Welche Hardware benötige ich für Satelliten Internet?

Alles was Sie benötigen ist eine Satellitenantenne und ein Astra Connect Modem.
Darüber hinaus ist keine zusätzliche Software auf Ihrem Rechner nötig. Sie müssen lediglich Ihren Rechner an das Modem anschließen, die Antenne installieren und exakt ausrichten lassen und schon haben Sie über Astra Connect einen vollwertigen und schnellen Zugang zum Internet. Ein Koaxialkabel zur Verbindung zwischen Modem und Antenne erhalten Sie von uns.

Hardware
Wie montiere ich meine Satelliten Anlage?

Es gibt drei Arten der Anbringung für die Satelliten Anlage. Wichtig ist immer: sie muss freie Sicht nach Süden haben, damit sie das Signal einwandfrei empfangen kann. Sie können die Novostream Anlage über einen Wandhalter an der Hauswand befestigen (vor allem für Wohnungen interessant), auf dem Dach Ihres Hauses befestigen oder mit einen mit Betonplatten beschwerten Terrassenständer auf das Garagendach stellen. Immer muss ein Kabel durch die Hauswand in den Innenraum führen. Die Satelliten Anlage wird ohne Montagematerial an Sie ausgeliefert, da die Umstände für die Montage bei jedem Kunden anders sind. In unserem Zubehörshop finden Sie aber das entsprechende Montagezubehör. Bitte beachten Sie, dass Novostream ausschließlich hochwertiges Montagematerial verkauft, weil eine sehr stabile Anbringung für ein störungsfreies Signal unabdingbar ist.

Benötige ich eine zusätzliche Telefonverbindung?

Nein, für einen Internet-via-Sat Anschluss von Novostream ist keine Telefonverbindungen erforderlich.

Kann ich meine aktuelle TV Sat Anlage für die Novostream Tarife benutzen?

Nein! Für den Novostream Anschluss benötigen Sie eine spezielle SAT-Schüssel. Die spezielle SAT-Antenne besteht aus einer Empfangseinheit UND einer Sendeeinheit. Sie können jedoch, nachdem Sie die Novostream Antenne montiert haben, ASTRA TV Kanäle mittels eines zusätzlichen LNB’s und eines Multifeedadapters nutzen.

Welche Computer benötige ich für die Nutzung von Internet via Satellit?

Sie nutzen alle Geräte, die sich mit dem Internet verbinden können. Das kann Ihr PC/Notebool sein. Neuerdings können auch mobile Geräte wie Smartphones und Tablets eine Verbindung zum Internet herstellen, jedoch nur noch kabellos, über ein sogenanntes W-LAN Netzwerk. Um das schnelle Internet aus dem All zu einem W-LAN Signal umzuwandeln, benötigen Sie einen Router. Hier gibt`s verschiedene Modelle in verschiedenen Preisklassen - ab 39,90 €. Alle Router, die Sie im Novostream Zubehörshop erhalten sind auf Kompatibilität mit den Novostream Produkten geprüft

Kann ich mit der Novostream SAT Anlage auch TV Programme empfangen?

Ja. Mit einem Zusatz-LNB (im Lieferumfang des Multifeedadapters enthalten) können Sie Satelliten TV-Programme empfangen und sehen, eine Verbindung mit dem Internet muss aber dafür nicht bestehen.
